# Price Action Skill for Manus

This repository contains a professional **Price Action Analysis Skill** designed for Manus agents. It integrates advanced trading methodologies to provide high-precision market analysis and trade setups.

## Overview

The **Price Action Skill** synthesizes multiple institutional-grade trading concepts:
- **ICT Methodology**: Liquidity pools (SSL/BSL), Fair Value Gaps (FVG), Displacement, and Market Structure (BOS/CHoCH).
- **GEX/Gamma Mechanics**: Dealer hedging dynamics, Call/Put walls, and Zero Gamma levels.
- **Classic Price Action**: Support/Resistance, Volume analysis, and Candlestick patterns.
- **Intraday Anchors**: VWAP, Pre-market levels, and Opening Range dynamics.

## Features

- **Structured Analysis**: Detailed breakdown of bias, key levels, and trade setups.
- **Community Content**: Optimized for sharing on platforms like Futu NiuNiu or Discord with a peer-to-peer voice.
- **Intraday Recaps**: Concise summaries of session behavior and volume context.

## How to Use

To use this skill with your Manus agent, you can:
1. Clone this repository.
2. Point Manus to the `skills/price-action/SKILL.md` file.
3. Ask Manus to analyze a chart or provide a trade setup using the Price Action framework.
